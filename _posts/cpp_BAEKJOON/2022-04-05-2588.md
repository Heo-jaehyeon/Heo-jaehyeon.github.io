# HINT

**백의 자리 숫자**를 구하려면 **나누기 연산자**를 사용해서

**(숫자) / 100** 를 하면 된다   



**십의 자리 숫자**를 구하려면 **나머지와 나누기 연산자**를 사용해서

**((숫자) % 100) / 10** 를 하면 된다   



**일의 자리 숫자**를 구하려면 **나머지 연산자**를 사용해서

**((숫자) % 100) % 10** 를 하면 된다   



# 소스코드

```c++
#include <iostream>
using namespace std;

int main()
{
	int A, B;
	cin >> A >> B;
	
	cout << A*((B%100)%10) << endl << A*((B%100)/10) << endl << A*(B/100)\
	<< endl << A*B << endl;
}
```
