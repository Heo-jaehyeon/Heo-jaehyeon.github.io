# HINT

**숫자 문자 0 부터 9까지는 아스키 코드가 연속적으로 48 부터 57로 이루어져 있다**

정수를 문자열로 받을때,

문자열 배열의 값은 아스키코드로 이루어져 있으므로

실질적인 값을 구하기 위해서는 

문자열 배열 원소에서 48 또는 아스키코드 48를 뜻하는 '0'를 빼면 된다   


   
etc) 아스키코드

대문자 알파벳 'A'~'Z'은 65~90

소문자 알파벳 'a'~'z'은 97~122

특수문자 32~47

제어문자 0~31



## 소스코드

```c++
#include <iostream>

using namespace std;

int main()
{
	int N, sum = 0;
	cin >> N;
	
	char A[]={0,};
	cin >> A;
	// 예시로 54321 를 A 배열에 넣는다면,
    // A[]={'5','4','3','2','1',\0} 이 저장되어 있다
    
	for (int i = 0; i < N; i++)
	{
		sum+=A[i]-48;
	}
	
	cout << sum << endl;
}
```
